Viewport(视口)

<meta name="viewport" content="width=device-width,initial-scale=1,user-scalable=no">
  1. width: 视口宽度，可写死一个固定像素值，让网页等比例缩放。
  2. initial-scale: 初始的缩放比例，取值范围0-10，一般为1。
  3. user-scalable: 是否允许用户缩放网页。yes或者no。

布局技巧

  1. 先写大容器，用HTML5标签表示。
  2. 选内容容器(container, container-fluid)(正常, 全屏)
  3. 在容器内写栅格。

栅格系统

规定任何屏幕下面都分12列，布局需要N列，即col-xx-(12/N)

<div class="row">
    <div class="col-xs-12 col-sm-6">No.1</div>
    <div class="col-xs-12 col-sm-6">No.2</div>
</div>
  1. 超小屏幕: col-xs-
  2. 平板屏幕: col-sm-
  3. 中等PC屏幕: col-md-
  4. 超大PC屏幕: col-lg-

几个注意事项

  1. 栅格class要从小屏幕写起，若大屏幕与最接近的小屏幕效果一致，可忽略。
  2. 若栅格本身就占12列，那不建议使用栅格系统，直接width:100%即可。
  3. 若栅格属于同一栏目，只要下一行足够高，不必另起row。
  4. 如果意识到一行内有个别列会高于其他列，记得在行末尾加上一个div.clearfix，这个.clearfix不能有高度或背景样式。

断点：768px 992px 1200px

列偏移

col-xs-offset-
  1. 偏移值会算在列数内。
  2. 只给要偏移的栅格加class即可。
  3. 居中: 偏移（12-栅格列数）/2

列嵌套

在一个栅格里嵌套一列，把这个栅格分成12列。

<div class="col-xs-12 col-sm-6">
    <div class="row">
        <div class="col-xs-2">Ha.1</div>
        <div class="col-xs-2">Ha.2</div>

        <div class="col-xs-2">Ha.5</div>

    </div>
</div>

UI组件

Glyphicon Icons组件

此UI 组件是用字体实现的，可以无限放大。
改变颜色用color，改变大小用font-size 

<i class="glyphicon glyphicon-XXX">

search 放大镜、star 实心五角星、star-empty 空心五角星、ok 钩、remove 交叉...

Button(按钮)组件

<button type="button" class="btn btn-XXX"></button>
<a href="#" class="btn btn-XXX"></a>
  1. default 白色
  2. primary 蓝色
  3. info 浅蓝色
  4. success 绿色
  5. warning 橙黄色
  6. danger 红色

若有需要可以加上.btn-lg让按钮变得更大。

导航条

  1. nav.navbar.navbar-XXX(default, inverse)
  2. 选container

商标部分（不隐藏或收缩部分）

此版本含伸缩按钮(data-target属性要与收缩容器的id一致)

<div class="navbar-header">
    <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#navigation">
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
    </button>
    <a href="index.html" class="navbar-brand">
        <img src="images/logo.png">
    </a>
</div>

链接部分（需隐藏或收缩的部分）

  1. 先写一个div.collapse.navbar-collapse，负责包裹整个要收缩的容器，给其定一个id。
  2. 主要链接部分写在一个ul.nav.navbar-nav内。（链接激活状态是给li标签加.active）
  3. 如有需要，可以添加导航栏表单组件，下拉菜单组件。切记加上.navbar-left或者.navbar-right
<div class="collapse navbar-collapse" id="navigation">
    <ul class="nav navbar-nav">
        <li class="active"><a href="">首页</a></li>
        <li><a href="">Laya家族</a></li>

    <form class="navbar-form navbar-left">
