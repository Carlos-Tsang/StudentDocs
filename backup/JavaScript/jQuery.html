jQuery框架

基础语法

$, jQuery //本质上都是一个函数

因为不排除$在其他插件中使用同名变量，为确保$是jQuery的，可在jq代码外加闭包。

(function($){
    //TODO
})(jQuery);

选择器

与CSS3选择器一致，作为参数传入$ / jQuery 函数内。

//html:
<button data-src="ert">Click Me!</button>
<button>Haha!</button>

//js:
var btn = $('button[data-src*=e]');
console.log(btn); //输出第一个button

注意事项：

通过$ / jQuery 选择的DOM，返回的是JQ对象，不可以直接在其后用原生JS语法。

//JQ对象 -> 原生JS节点
$('button[data-src*=e]')[0]; //在$函数后加[0]

同样，原生JS节点不可使用JQ语法，必须转换JQ对象方可使用。

//原生JS节点 -> JQ对象
$(node); //$函数包裹节点对象即可

每次用JQ选择器，建议用变量接收，后续用该变量操作。

//每个JQ对象都支持链式调用，如：

事件冒泡

有时候难免前辈元素有点击事件，元素本身又要添加相同的事件类型，这样会触发2种事件，这时给元素本身加事件冒泡，使其触发事件时与前辈元素事件分离。

$('body').click(function(){
    console.log('bodyHandler!!');
});

$('h1').click(function(event){
    event.stopPropagation();
    console.log('H1handler!!');
});

用法可参考上面代码。

animate(Object, interval, f(n))（自定义动画）

使用方法：

  1. 先给执行动画的对象添加默认css属性。（常见：position:relative;）
  2. 在Object内写上动画执行完毕的css样式。（一些临界值明显的属性可用'toggle'表示，如:height,width, opacity）
  3. 添加interval过渡时间。
  4. 若有需要，添加回调函数f(n)，在动画执行完毕后调用。
var pwd = $('#password'),
    form = $('form');
pwd.on('keydown', function(event){
    if(event.keyCode == 13){
        h1.animate({
            left: 400,
            width: 'toggle'
        }, 'normal', function(){
            form.submit();
        });

    }
})

stop() 停止动画

直接在执行动画的对象调用.stop() ，可终止动画效果。

delay(interval) 延时动画

在动画之前或两个动画之间加.delay(interval) ，可等待一段时间再执行动画。

注意事项： delay()函数后面不一定要接动画类函数。

var h2 = $('h2');

遍历

//html
<div id="div1">
    <div id="div2">
        <h2>
            <p>123</p>
        </h2>
    </div>
</div>

向下遍历(children, find)

children(), children('filter') 遍历子辈

var div1 = $('#div1');
console.log(div1.children()); //[div#div2]，实际情况不止一个，是全部子辈
console.log(div1.children('div')); //[div#div2]
console.log(div1.children('h2')); //[]

find('filter') 一直往子辈方向遍历，空为止。

console.log(div1.find('h2')); //[h2]

向上遍历(parent, parents, parentsUntil)

parent() 返回父节点

para.parent(); //[h2]

parents(), parents('filter') 一直往父辈方向遍历，直到html

para.parents(); //[h2, div#div1, div#div2, body, html]
para.parents('div'); //[div#div1, div#div2]

parentsUntil('selector') 当前节点往父辈方向遍历，到selector节点为止，返回对象不包括selector本身。（开区间）

para.parentsUntil('#div2').css('border', '3px solid chocolate');

同级遍历(siblings, next, nextAll, nextUntil, prev, prevAll, prevUntil)

siblings(), siblings('filter') 获取所有同级节点

//html
<div class="banner">
    <button>Btn1</button>
    <span>123</span>
    <button>Btn2</button>
    <span>123</span>
    <button>Btn3</button>
</div>

//js
var $btns = $('.banner button'); 

$btns.first().addClass('bgGreen');

$btns.on('click', function(){
    $(this).addClass('bgGreen').siblings('button').removeClass('bgGreen');
});

next(), nextAll(), nextAll('filter'), nextUntil('selector')

next                下一个节点
nextAll             所有往下遍历的兄弟节点
nextUntil('selector')   所有往下遍历，直到selector为止的所有兄弟节点

var test = $para.first().nextUntil('#twelve');
console.log(test);

prev(), prevAll(), prevAll('filter'), prevUntil('selector')
用法同上，只是prev是向上遍历。

筛选器（filter, not, eq, first, last）

filter('selector') 在一堆jQ对象里过滤出符合selector类型的节点。

var test = $para.first();
console.log(test.parents().filter('#div2'));
console.log(test.parents('#div2')); //执行结果相同

not('selector') 过滤出符合selector以外的节点。

console.log(test.parents().not('#div2'));

eq(index) 返回jQ对象数组下标为index的节点。

console.log(test.parents().eq(1));

first(), last() 第一个子节点, 最后一个子节点。

console.log(test.first()); //第一个子节点
console.log(test.last()); //最后一个子节点
