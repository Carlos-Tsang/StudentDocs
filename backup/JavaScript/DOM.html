DOM基本操作

DOM对象获取

常规获取方法:

document.getElementById('id') //只输出第一个dom元素 document.getElementsByTagName('tagName') //输出一个数组 document.getElementsByName('name') //输出一个数组

Html 5 获取方法:

document.querySelector('Selector') //选择器与CSS完全一样，输出第一个 document.querySelectorAll('Selector') //同上，但是会输出一个数组

注意事项：如果一个DOM元素有id属性，可直接用id属性值作为JS-DOM对象使用。

DOM对象创建

document.createElement('tagName') //tagName是标签名

DOM节点内容

.innerHTML //节点内容,如果内容含引号，要在引号前加反斜杠(\)，表示转义字符。

添加DOM节点

a.appendChild(DOM) //在a里追加一个DOM a.cloneNode([true,false]) //克隆一个a节点，true代表深度克隆 document.body.insertBefore(a1,a);//在body元素里面, a之前添加a1

在DOM元素内第一个子节点前加一个新节点

function prependChild(parent, newChild){
    if(parent.firstChild){
        parent.insertBefore(newChild, parent.firstChild);
    } else {
        parent.appendChild(newChild);
    }   
}

移除DOM节点

a.removeChild(DOM) //在a里面移除一个DOM

节点指针

parent.firstChild, parent.firstElementChild(第一个元素节点)

代表parent里面的第一个子节点，注意空白文本节点（parent标签结束后的空格或换行会生成一个空白的文本节点）。

清除空白文本节点

function cleanWhitespace(oEelement){
    for(var i=0;i<oEelement.childNodes.length;i++){
            var node=oEelement.childNodes[i];
            if(node.nodeType==3 && !/\S/.test(node.nodeValue)){
            node.parentNode.removeChild(node)
            }
    }
}

parent.lastChild, parent.lastElementChild(最后一个元素节点)

代表parent里面的最后一个子节点，注意事项同上。

parent.childNodes

返回一个所有子节点的数组，包括空白文本节点。

brother.previousSibling, brother.previousElementSibling(前一个元素节点)

返回brother节点前一个兄弟节点

brother.nextSibling, brother.nextElementSibling(前一个元素节点)

返回brother节点下一个兄弟节点

child.parentNode

返回child的父节点

属性操作

获取对应属性值

node.getAttribute('属性名');

添加 / 设置对应属性值

node.setAttribute('属性名', '属性值');

移除对应属性

node.removeAttribute('属性名');
